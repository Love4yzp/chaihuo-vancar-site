# Design | 设计源文件

此目录包含所有可编辑的设计源文件。

## 目录结构

```
design/
├── mechanical/   # 机械 CAD 源文件 (STEP, Fusion 360, SolidWorks)
└── electrical/   # 电气原理图源文件 (KiCad, EasyEDA)
```

## 说明

- 这里存放的是**源文件**，可以直接编辑修改
- 导出的制造文件（STL、Gerber 等）请放在 `fabrication/` 目录
- 详细设计说明请参考 [语雀文档](https://yuque.com/chaihuo-vancar)

## 软件要求

- **机械设计**: Fusion 360 / SolidWorks / FreeCAD
- **电气设计**: KiCad 8.0+ / EasyEDA
