# Operation | 运行代码

此目录包含让基地车系统运行的所有代码。

## 目录结构

```
operation/
├── firmware/   # 嵌入式固件 (Arduino, ESP32, STM32)
└── software/   # 上位机软件 (服务器, IoT, 控制程序)
```

## Firmware 固件

控制车载设备的嵌入式代码：

- 电源管理
- 传感器采集
- 通信模块

### 开发环境

- Arduino IDE 2.x 或 PlatformIO
- ESP-IDF (如使用 ESP32)

## Software 软件

服务端和控制软件：

- IoT 数据采集
- 远程监控
- 设备配置工具
